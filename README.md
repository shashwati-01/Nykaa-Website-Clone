# Nykaa Website Clone

# Live Demo
https://shashwati-01.github.io/Nykaa-Website-Clone/

A frontend clone of the Nykaa beauty shopping website built using HTML and CSS.

The project recreates a beauty e-commerce homepage with navigation menus, promotional banners, brand sections, product categories, beauty advice sections, and a structured footer.

## Features

- Nykaa-inspired navigation bar
- Beauty category menu
- Search bar interface
- Promotional beauty banners
- Pink Friday sale section
- Top brands section
- Only at Nykaa section
- Makeup category showcase
- Special curations section
- Beauty advice section
- Structured website footer

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome
